## 一句话总结

这次更新为桌面应用添加了**供应商图标**，提升了视觉体验，并增强了 GitHub Actions 的安全性。

## 新功能：供应商图标

### 这次更新了什么？

桌面应用中的模型选择界面现在会显示各个 AI 供应商的图标，更直观地区分不同服务商。

### 有什么好处？

之前：
- 模型列表只有文字，不容易区分
- 需要记住每个模型的供应商

现在：
- 一眼就能看到是哪个供应商的模型
- 视觉体验更好，操作更方便

### 如何使用？

无需任何操作，图标会自动显示：

1. 打开桌面应用
2. 点击模型选择按钮
3. 现在可以看到每个模型的供应商图标

---

## 问题修复：GitHub Actions 安全性

### 之前的问题

OpenCode 生成的 GitHub Actions 工作流模板可能会泄露你的 GitHub 凭证（如果使用了 actions/checkout）。

### 现在的情况

所有工作流模板现在都添加了 `persist-credentials: false` 参数，防止凭证泄露。

**示例**：

```yaml
- uses: actions/checkout@v4
  with:
    persist-credentials: false  # ← 新增
```

### 对你有什么影响？

如果你使用 OpenCode 生成 GitHub Actions 工作流，现在会更安全，不用担心凭证被意外保存或泄露。
